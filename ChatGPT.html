give me a code of navigation bar of fortfoliob website with boiler plate

Certainly! Below is a simple HTML boilerplate with a basic navigation bar for a portfolio website. This example includes basic styling using CSS and responsive design considerations.

HTML Boilerplate

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Portfolio Website</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
        }
        .navbar {
            display: flex;
            justify-content: space-between;
            align-items: center;
            background-color: #333;
            padding: 10px 20px;
        }
        .navbar a {
            color: #fff;
            text-decoration: none;
            padding: 14px 20px;
        }
        .navbar a:hover {
            background-color: #ddd;
            color: black;
        }
        .navbar .logo {
            font-size: 1.5em;
            font-weight: bold;
        }
        .navbar .nav-links {
            display: flex;
        }
        @media (max-width: 768px) {
            .navbar .nav-links {
                display: none;
                flex-direction: column;
                width: 100%;
            }
            .navbar a.toggle-nav {
                display: block;
                font-size: 1.2em;
            }
            .navbar a.toggle-nav:hover {
                background-color: transparent;
            }
        }
    </style>
</head>
<body>
    <header>
        <nav class="navbar">
            <div class="logo">
                <a href="#">MyPortfolio</a>
            </div>
            <a href="javascript:void(0);" class="toggle-nav" onclick="toggleNav()">☰</a>
            <div class="nav-links" id="navLinks">
                <a href="#home">Home</a>
                <a href="#about">About</a>
                <a href="#projects">Projects</a>
                <a href="#contact">Contact</a>
            </div>
        </nav>
    </header>

    <script>
        function toggleNav() {
            var navLinks = document.getElementById("navLinks");
            if (navLinks.style.display === "flex") {
                navLinks.style.display = "none";
            } else {
                navLinks.style.display = "flex";
            }
        }
    </script>
</body>
</html>

Explanation

  1. HTML Structure:

    • The <header> element contains the <nav> element for the navigation bar.
    • The navigation bar has a logo, navigation links, and a toggle button for smaller screens.
    • The nav-links div contains the actual links to different sections of the portfolio.

  2. CSS Styling:

    • The body is styled to remove default margin and padding, and set a default font.
    • The .navbar class styles the navigation bar with a background color, padding, and spacing.
    • The .navbar a styles the links with color, padding, and hover effects.
    • The .logo class styles the logo with a larger font size and bold text.
    • The media query hides the navigation links and shows a toggle button on smaller screens.

  3. JavaScript:

    • The toggleNav() function toggles the display of the navigation links when the toggle button is clicked.

This code provides a basic and responsive navigation bar for a portfolio website. You can further customize the styles and functionality as needed.
